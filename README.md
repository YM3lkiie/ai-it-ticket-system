# AI IT Ticket System
Personal project to build an AI-powered IT support ticket platform.

# SET UP INSTRUCTIONS 

1. Clone the repository
git clone https://github.com/YM3lkiie/ai-it-ticket-system.git

2. Install dependencies
npm install

3. Create environment file
Copy .env.example and rename it to .env

4. Configure your database credentials

5. Initialize database
mysql -u root -p < database/init.sql

6. Start the server
npm run dev
